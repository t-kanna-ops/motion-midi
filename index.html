Motion MIDI Controller Web — 起動方法

カメラ・WebMIDI を使うには HTTPS または localhost から開く必要があります。

ローカルサーバーを起動してください：

npx serve .   または   python -m http.server 8080

その後 http://localhost:8080 (または http://localhost:5000) をブラウザで開いてください。

⚠ MIDI送信には Chrome / Edge を推奨します（WebMIDI対応）。
